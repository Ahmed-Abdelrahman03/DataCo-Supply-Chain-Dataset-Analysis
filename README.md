# DataCo Smart Supply Chain: Risk & Performance Analysis

##  Overview
This project focuses on analyzing a complex supply chain dataset to identify logistics inefficiencies, specifically focusing on **Late Delivery Risks** and their **Financial Impact** on global markets.

##  Key Insights
* **High-Risk Zones:** LATAM and Europe identified as regions with the highest late delivery frequency.
* **Logistics Bottleneck:** 'Standard Class' shipping accounts for the majority of delays.
* **Business Impact:** Correlated shipping delays with a significant drop in net benefit per order.

##  Tools Used
* **SQL (BigQuery/MySQL):** Data cleaning, transformation, and complex querying.
* **Tableau:** Interactive dashboarding and storytelling.
* **Excel:** Initial data exploration and validation.
